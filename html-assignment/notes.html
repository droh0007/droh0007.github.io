Tags that I've learned

<P>
This tag is used to add paragraphs of text.
<EM>
This tag stands for emphasis and is how we italicize text.
<Strong>
This tag will make the text contained bold.
<Blockquote>
This tag allows us to have stand alone quotes.
<Address>
This tag is where you would put a physical address and contact info.
